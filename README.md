## 🚀 Features

### 🔎 Smart Search

* Search **movies, TV series, anime, and actors**
* Auto suggestions while typing

### 🎬 Movie Details

* Poster
* Title
* ⭐ Rating
* Overview

### 🎥 Trailer

* Watch official trailer directly inside app

### 📺 OTT Platform Detection

* Shows availability on:

  * Netflix
  * Amazon Prime
  * Disney+ Hotstar
  * Others

### 👨‍🎤 Actor Search

* Search actors like:

  * Yash
  * Prabhas
  * Allu Arjun
* Displays their movies

### 🔥 Trending Homepage

* Displays trending movies & series

### 🎞 Interactive UI

* Click any movie → opens full detail page
* Smooth hover animation

## ⚙️ Installation & Setup

### 1. Clone Repository

git clone https://github.com/your-username/movie-recommender-system.git
cd movie-recommender-system

### 2. Install Dependencies

pip install -r requirements.txt

### 3. Add TMDB API Key

Open `movie_api.py` and replace:

API_KEY = "YOUR_API_KEY"

With your key:

API_KEY = "your_tmdb_api_key"

### 4. Run App

streamlit run app.py
